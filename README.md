# Thetapad Community Feedback

Welcome to the official feedback repository for [Thetapad](https://thetapad.com) - the privacy-first LaTeX editor.

## What is this repository?

This is where you can:

- **Report bugs** - Found something broken? Let us know!
- **Request features** - Have an idea to make Thetapad better? We'd love to hear it!
- **Ask questions** - Need help with something? Ask away!
- **Discuss ideas** - Join conversations about the future of Thetapad

## How to contribute

1. **Search first** - Check if your issue or idea has already been reported
2. **Use templates** - Click "New Issue" and select the appropriate template
3. **Be descriptive** - The more details you provide, the better we can help

## About Thetapad

Thetapad is a privacy-first LaTeX editor that compiles documents locally in your browser. Your documents never leave your device.

**Key features:**
- Client-side compilation (no server uploads)
- Real-time collaboration via P2P
- Works offline
- AI-powered error explanations
- Visual table builder
- Smart citations
